# FreeCell Solver – Python dependencies
# All dependencies are part of the Python 3 standard library.
# No third-party packages are required.
#
# Minimum Python version: 3.9
#
# Standard-library modules used:
#   tkinter     – GUI (included with most Python distributions)
#   collections – deque for BFS
#   heapq       – priority queue for UCS and A*
#   threading   – background solver execution
#   tracemalloc – memory measurement
#   time        – timing
#   random      – random deal generation
#   dataclasses – Move and SolverResult data containers
